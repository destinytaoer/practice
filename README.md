- lazyLoad：懒加载

## utils
- toJson：将 JSON 格式字符串转化为 JSON 对象
- listToAry：将类数组转化为数组

## DOM
- DOM：封装了类似 JQ 中的 DOM 操作方法

## 动画
- animate0：简单的移动动画
- animate：较完善的移动动画

## 发布订阅
- subscribe：单个容器的订阅发布
- subscribes：支持多个容器的订阅发布

## ajax
- ajax：使用原生 JS 封装的 ajax 方法
- ajax-promise：基于 Promise 封装的 ajax 方法

## Promise
- promise：使用原生 JS 手写的简易 Promise 类
